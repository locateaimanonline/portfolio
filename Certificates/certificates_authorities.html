Managing Certificates

Manage and deploy digital certificates in SOTI MobiControl, including manual uploads and dynamic certificates issued via integrated certificate authorities.

SOTI MobiControl enables you to manage manually uploaded (external certificates) or integrate certificate authorities to issue managed certificates using predefined templates. This section includes the following topics:

  • Deploy manually uploaded certificates. See Deploying Manually Uploaded Certificates
  • Deploy managed certificates. See Deploying Managed (Dynamic) Certificates.
    1. Step One: Integrate Certificate Authority Services.
    2. Step Two: Create a Certificate Template.
    3. Step Three: Deploy Dynamic Certificates.
  • Manage installed certificates on a device. See Managing Certificates on a Device.
  • Review and manage certificates from the SOTI MobiControl web console. See Using the Certificates Dashboard.

The Device Tab

The Device tab in the device information panel displays key properties and system details for the selected device. Administrators can use this tab to review essential device information such as hardware specifications, OS version, enrollment status, and more. See Viewing Device Information for instructions on accessing this panel.

Tip: Use the search bar to quickly locate specific device properties or values within the Device tab.

Feature Availability

Specific features depend on the SOTI MobiControl version and device type:

  • VPN IPv4 and VPN IPv6 details require SOTI MobiControl 15.4.3 or later.

  • Corporate Owned Personally Enabled (COPE) devices do not support VPN details.

  • BitLocker Recovery Key details require additional steps.

The Device Details tab organizes information into the following sections based on platform compatibility.

Section Details Platform
Antivirus Displays Windows Defender configuration and scan details:
  • Sync: Use the Sync button to fetch the latest antivirus details.
  • Last Synced On: Shows the last successful sync.
  • Scan History: Displays timestamps for the previous quick and full scans.
  • Active Threats: Indicates unresolved security risks.
    • True: Windows Defender requires manual intervention.
    • False: Windows Defender has mitigated detected threats.
  • Quarantined Threats: Displays the number of quarantined threats.
  • Threat History: Provides access to the entire history of detected threats.
Important: Antivirus details do not update automatically after upgrading SOTI MobiControl. Sync manually before reviewing data.

Network Displays network connection and configuration details:
  • IP Address & IPv6 Address: Shows current Wi-Fi network addresses.
  • Network Type: Displays whether the device is on Wi-Fi, Ethernet, or Cellular.
  • Network SSID: Identifies the connected Wi-Fi network.
  • Network Authentication: Displays authentication protocols (for example, WPA2, WPA3).
  • Subnet Mask: Specifies network segmentation details.
  • Wi-Fi & Wireless LAN Modes: Indicates whether the device uses infrastructure or ad hoc modes.
  • ICCID: Displays the SIM card identifier.

  • WIFI Root Certificate Serial Number: Displays the serial number of the active Wi-Fi root certificate.

  • WIFI Root Certificate Expiry Date: Shows the expiry date of the active Wi-Fi root certificate.

  • WIFI User Certificate Serial Number: Displays the serial number of the active Wi-Fi user certificate.

  • WIFI User Certificate Expiry Date: Shows the expiry date of the active Wi-Fi user certificate.

  • Proxy URL: Displays the proxy URL the device connects to.

  • Proxy Host: Shows the proxy host the device connects to.

  • Proxy Port: Displays the proxy port the device uses to connect.

Note: This requires Configure devices/device groups permission. The console shows the last known network details if the device is offline.

The Configurations Tab

The Configurations tab in a Windows Modern device’s Information Panel provides an overview of the device’s configuration status. It includes details about the profiles and packages applied to the device. See Viewing Device Information for instructions on accessing this tab.

Profiles

This section displays the profiles assigned to the device, including installation status and protocol type.

Column Description
Profile Name The name of the assigned configuration profile.
Status
  • Installed: Profile installed successfully.
  • Installed Partially: Profile installed partially due to unmet or unknown activation conditions.
    Note: For Apple devices, this occurs when a compatible Apple device uses a declarative profile with an unmet or unknown Activation condition.
  • Failed: Profile installation failed.
  • Pending Install: The profile is scheduled for installation.
Install Method Details whether the profile is installed automatically or manually.
Version Displays the version number of the assigned profile.
Assignment Date Shows the time stamp when the profile was assigned to the device.
Protocol Indicates whether the assigned profile uses the MDM (reactive) or DDM (declarative) protocol for management.

Package Details

This section lists software packages deployed to the device.

Column Description
Package Name The name of the package installed on the device.
Version Displays the package version number.
Installation Order Indicates the order in which the package was installed.
Referenced Profile Shows the profile associated with the package.
Status Indicates whether the package is Installed or is in another state.

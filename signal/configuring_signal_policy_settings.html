Signal Policy Wizard | Configure Policy Settings

Ensure you have Manage Signal Policies permissions. You must define Signal policy conditions before configuring the settings. See Signal Policy Wizard | Build Policy Conditions.
This is the third of five steps in configuring a Signal policy. In this step, you configure policy settings that control how conditions are evaluated, including device-side evaluation and location-based power policies. You also define their frequency of evaluation.

Depending on the selected condition type, SOTI MobiControl can configure supported devices to evaluate Managed Device properties at a defined frequency. Device-side evaluation allows policies to respond proactively on the device, rather than relying on server-side evaluations triggered by device check-ins.

You can also configure a device count trigger to ensure that actions are executed only when a specific number of devices meet the defined conditions within a given time span.

For an overview of all five steps in the Signal policy workflow, see Signal Policy Wizard Overview.
Restriction:
  • Device-side evaluation is supported only for specific Managed Device properties. For the complete list, see Device-Side Evaluation Properties.
  • The Outdoor Geofence Power Policy is available only when using the following event attributes:
    • Enter Outdoor Geofence
    • Exit Outdoor Geofence
  1. Toggle the Enable Device Side Evaluation option.
  2. Select an evaluation frequency for the policy conditions.
  3. Optional: Turn on Enable device count trigger and configure the trigger conditions.
    • Number of devices targeted: Specify the minimum number of devices that must meet the conditions.
    • In the span of: Specify a duration value and select a time unit (for example, hours or days).
  4. Optional: If the policy uses Enter Outdoor Geofence or Exit Outdoor Geofence, configure the Outdoor Geofence Power Policy.
    You can choose one of the following options:
    • Continuous: Continuously checks the device location. Uses more battery power.
    • Periodic: Checks the device location at a configured interval. Specify a value and select one of the following periods:
      • Minutes
      • Hours
      • Days
      • Weeks
Select Next to continue to the Actions tab. See Signal Policy Wizard | Add Policy Actions.

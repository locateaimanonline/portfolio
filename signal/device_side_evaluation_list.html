Device-Side Evaluation Properties

About This Reference

Use this reference when configuring Signal policy settings to understand which properties are eligible for device-side evaluation on each supported platform. For procedural instructions, see Signal Policy Wizard | Configure Policy Settings.

It lists the Managed Device properties that support device-side evaluation, which determines whether conditions are evaluated directly on the device instead of on the server. The following table lists configurable Managed Device properties that support device-side evaluation, along with platform availability.

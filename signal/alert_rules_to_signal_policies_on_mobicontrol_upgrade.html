Legacy Alert Rules Now Signal Policies

Starting SOTI MobiControl version 2025.0, SOTI deprecates legacy Alert rules and automatically converts them to Signal policies. An announcement detailing this conversion has been sent to administrators.

Viewing Converted Alerts

Check the announcements to see which legacy Alerts were successfully converted and which failed.
Note: Legacy Alert rules fail to convert when a configuration, attribute, or operator is not supported in Signal.
  1. Select the bell icon on the top-right of the web console, then select Announcements.
  2. In the Announcements section, look for the Legacy Alerts Status notification, then select More > View Converted Alerts to view the Converted Legacy Alerts window.
  3. Optional: In the Converted Legacy Alerts window, select Download Detailed Report to get a .csv file with comprehensive details of all previously configured legacy Alert rules. This report also explains why some legacy Alert rules may have failed to convert.

Post-conversion Actions

Successfully converted policies appear in the Signal policies section, initially in a disabled status. Administrators can review and re-enable these policies as needed. You can access this by navigating to Policies > Signal from the main menu. You can also navigate to the Signal policies section in the Converted Legacy Alerts window by selecting Go to Signal Policies.

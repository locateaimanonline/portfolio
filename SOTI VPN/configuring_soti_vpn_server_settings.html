Configuring SOTI VPN Server Settings

You must have Manage SOTI VPN permissions to configure SOTI Virtual Private Network (VPN) Advanced Configurations settings.
After enrolling your SOTI VPN servers (see Enrolling SOTI VPN Servers), use the SOTI Virtual Private Network (VPN) Advanced Configurations settings to enable SOTI VPN servers and configure an out-of-contact threshold for VPN client communications.
  1. From SOTI MobiControl, open the Devices view.
  2. From the list of devices, select a SOTI VPN server. A window listing its device properties opens.
  3. Navigate to the Configurations tab and scroll to the Advanced Configurations section.
  4. From the Advanced Configurations list, select SOTI VPN Server Settings. The SOTI VPN Server Settings window opens.
  5. Optional: Enable Override Settings Inherited From Parent Group.
  6. Toggle on Enable VPN Server to enable VPN servicing capabilities on the selected server.
  7. Optional: Set an Out of Contact Threshold value to specify compliance requirements for its clients. If a device remains disconnected for longer than the specified period from SOTI MobiControl, VPN services are no longer offered until it reconnects. After reconnecting, the new key may take a few minutes to propagate across the SOTI VPN servers.
This brings up the VPN interface on the server, configuring clients to use it to access enterprise network resources.
Set up your VPN clients. See Configuring SOTI VPN Clients.
